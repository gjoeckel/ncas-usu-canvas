# Hard Refresh in Chrome on Windows

## Quick Method (Keyboard)
Press **Ctrl+Shift+R** or **Ctrl+F5**

## Alternative Method (DevTools)
1. Press **F12** to open DevTools
2. Right-click the **Refresh button** (circular arrow)
3. Select **"Empty Cache and Hard Reload"**

## If Still Not Working
1. Press **F12** to open DevTools
2. Go to **Network** tab
3. Check **"Disable cache"** (checkbox at top)
4. Press **F5** to refresh

This forces Chrome to fetch fresh copies of your CSS and JS files from the local server.
